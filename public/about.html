TAKT = Customer Needs Products Qty / Time Manufacture has
Example:
TAKT = 480 pcs / (60*60*8) seconds => 60 seconds / pcs
Cycle Time = A time of period to make one product
Example:
Cycle Time = 100 pcs / 30000 seconds => 300 seconds / pcs
Labor plan: How to meet customer takt for product delivery.
Example:
Labor needs = 300 seconds / 60 seconds => 5 labors / pcs

Production Standard
Piece / People / Hour (PPH) : Products made per people per hour
Example:
PPH = 60 pcs / (5 people * 8 hours ) => PPH = 1.5
Productivity Evaluation
Time Measurement: Standard Time Spent / Actual Time Spent * 100%
(Standard Time = Output / PPH)
Output Measurement: Actual Output / Standard Output * 100%
(Standard Output = Time Spent * PPH)
Example: If production plan to make 480 pcs. Actual time spent is 328 hours.
Standard Time Spent should be = 480 pcs / 1.5 => 320 Hours
Achieve % = 320 / 325 = 98.5%
Example: If production plan to work 320 Hours. Actual output is 473 pcs
Standard Output should be = 320 Hours * 1.5=> 480 Pcs
Achieve % = 473 / 480 = 98.5%
