Labor plan. TAKT & Cycle time.

PPH stands for pcs per people per hour. PPH is the baseline for measuring workforce performance.

Productivity evaluation. Time measurement or ouptut measurement.

Time Measurement: Standard Time Spent / Actual Time Spent * 100% (Standard Time = Output / PPH). Example: If production plan to make 480 pcs. Actual time spent is 50 hours. Standard Time Spent should be = 480 pcs / 12 => 40 Hours Achieve % = 40 / 50 = 80%
